# Legacy Chess Materials

This area collects historical chess-study materials from the original project.

## Files Here

- `papers/original_model.pdf`: original confidence-model paper.
- `papers/data_paper.pdf`: original data/experiment paper.

## Related Files Still Kept at Root

The following folders are still kept in their original locations because many
analysis scripts use hard-coded paths:

- `Data/`: legacy chess participant CSV files.
- `Code/`: original MATLAB scripts, `.mat` files, and derived CSV files.

Do not move `Data/` or `Code/` until the scripts have been updated to use a
central path configuration.
