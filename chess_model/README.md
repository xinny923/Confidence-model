# Chess Model

This area documents the improved confidence model developed from the chess
dataset.

## Files Here

- `papers/my_paper.pdf`: conference paper based on the improved model.

## Related Files Still Kept at Root

The main chess-model scripts and outputs are currently kept in their original
locations so existing relative paths continue to work:

- `benchmark.py`: original-model Python reproduction on chess data.
- `original_common.py`: shared original-model utilities.
- `Paper_fig/`: improved model scripts, model variants, paper figures, and
  output folders.
- `outputs_extended/`: earlier extended-model outputs.

Future cleanup can move these into this folder after path handling is
centralized.
